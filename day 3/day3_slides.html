Vocabulary Time! 📚

Gigabyte (GB)
A unit of digital storage equal to approximately 1 billion bytes (technically 1,024 megabytes). Think of it like a large box that can hold about 250 songs or a 30-minute HD video.

Fun Fact: The first hard drive in 1956 could store 5 MB and weighed over a ton! Your phone now has thousands of times more storage and fits in your pocket!

The Command Line 🖥️

Before we had mice and touchscreens, we talked to computers by typing commands!

Mac (Terminal)

$ pwd
/Users/student/Desktop
$ ls
homework.txt games pictures

Windows (Command Prompt)

C:\> cd
C:\Users\student\Desktop
C:\> dir
homework.txt games pictures

Cool Fact: Hackers in movies always use the command line because it looks cooler (and it's actually faster for many tasks!)

Vocabulary Time! 📚

Command Line (Terminal)
A text-based way to control your computer by typing commands instead of clicking with a mouse. On Mac, it's called Terminal. On Windows, it's called Command Prompt or PowerShell. It's like texting your computer instructions!

Think of it like this: GUI (clicking) is like using Uber's app, Command Line is like texting the driver directly!

Essential Commands 🎯

Action Mac (Terminal) Windows (Command Prompt)
Where am I? pwd cd (no arguments)
List files ls dir
Change folder cd foldername cd foldername
Create folder mkdir newfolder mkdir newfolder
Clear screen clear cls

Vocabulary Time! 📚

HTML
HyperText Markup Language - The standard language for creating web pages. It uses tags (like labels) to tell the browser what each part of the page is: headings, paragraphs, images, links, etc. It's the skeleton of every website!

Fun Fact: HTML was invented in 1991, before Google, Amazon, or even Yahoo existed!

Your First HTML 🏗️

<p>This is my first web page!</p>
<button>Click Me!</button>

This is my first web page!

HTML Tags Explained 🏷️

HTML tags are like labels that tell the browser what something is:

<h1> ... </h1>
Main heading (biggest)
<p> ... </p>
Paragraph of text
<img src="picture.jpg">
Display an image
<a href="google.com"> ... </a>
Create a link

Complete HTML Page 📄

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my website.</p>
  </body>
</html>
