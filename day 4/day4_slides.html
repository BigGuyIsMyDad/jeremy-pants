Another Vocabulary Word 📚

Git
A version control system that tracks changes in your code. Think of it like a super-powered "undo" button that remembers every change you've ever made to your files!

Key Git Concepts:

  • 📸 Commit = A snapshot of your code at a specific point
  • 🌳 Branch = A separate version to try new ideas
  • 🔄 Push/Pull = Send/receive changes to/from GitHub
  • 🍴 Fork = Make your own copy of someone's project

📝 Git is different from GitHub! Git is the tool, GitHub is a website that uses Git.

What is HTML? 🌐

HTML = HyperText Markup Language

It's the language that creates websites! Think of it like:

  • 📝 The skeleton of a webpage
  • 🏗️ Building blocks that browsers understand
  • 🎨 The structure that holds content
<!-- This is what HTML looks like -->
<h1>Hello World!</h1>
<p>This is a paragraph.</p>

Every website you've ever visited started with HTML!

12 / 26

Basic HTML Structure 🏗️

Every HTML page has this basic structure:

<!DOCTYPE html>
<html>
<head>
<title>My Page Title</title>
</head>
<body>
<h1>Welcome to My Page!</h1>
<p>This is where your content goes.</p>
</body>
</html>

Think of it like a sandwich:

  • 🍞 <html> = The bread that holds everything
  • 🧠 <head> = Info about your page (not visible)
  • 🥪 <body> = The actual content people see

Install Git 🔧

Git = The tool that tracks changes to your code

For Windows:

  • Download from git-scm.com
  • Run the installer (use default settings)
  • Open "Git Bash" when done

For Mac:

  • Open Terminal (find it in Applications → Utilities)
  • Type: git --version
  • If not installed, it will prompt you to install
20 / 26

Configure Git ⚙️

Tell Git who you are (use Git Bash on Windows, Terminal on Mac):

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Use the same email you used for GitHub!

📝 Replace "Your Name" and email with your actual info!

Commit Your Changes 💾

Now let's save your changes to Git:

git add .

Create a commit (a save point):

git commit -m "Add my webpage to students folder"

Push to GitHub:

git push

📝 You might need to login to GitHub when you push!

25 / 26

Create a Pull Request 🚀

Pull Request (PR) = Asking to merge your changes into the original project

1 Go to YOUR fork on GitHub

2 Click "Contribute" → "Open pull request"

3 Add a title: "Add [Your Name]'s webpage"

4 Add a description of what you did

5 Click "Create pull request"

🎉 Congratulations! You're now a GitHub contributor!
