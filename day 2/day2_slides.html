üìö Today's Vocabulary

1. Operating System (OS)
The software that manages computer hardware and provides services for programs. It's like the manager of a restaurant - coordinating everything!
2. File Extension
The suffix at the end of a filename that tells the computer what type of file it is (.txt, .jpg, .exe)
3. Directory/Folder
A container that holds files and other folders - like a filing cabinet drawer
4. Path
The address showing the location of a file or folder - like a street address for your files
5. RAM vs Storage
RAM: Fast temporary memory (lost when power off)
Storage: Slower permanent memory (keeps data after power off)

üíæ Understanding Storage Sizes

The Storage Ladder

8 bits = 1 Byte (a single character)

1,000 Bytes = 1 Kilobyte (KB) - A short email

1,000 KB = 1 Megabyte (MB) - A song

1,000 MB = 1 Gigabyte (GB) - A movie

1,000 GB = 1 Terabyte (TB) - 250 movies!

üß† RAM vs Storage

üöÄ RAM

Speed: Super Fast!

Capacity: 8-32 GB typical

Persistence: Lost when power off

Like: Your desk while working

Use: Running programs

üíΩ Storage (SSD/HDD)

Speed: Slower than RAM

Capacity: 256 GB - 2 TB typical

Persistence: Keeps data forever

Like: Your filing cabinet

Use: Saving files

Think of it this way: RAM is like your brain's short-term memory (what you're thinking about right now), Storage is like long-term memory (what you learned last year)

Basic Commands:

  • cd - Change Directory (move between folders)
  • ls - List (show what's in a folder)
  • mkdir - Make Directory (create a folder)
  • echo - Print text
  • cat - Display file contents
